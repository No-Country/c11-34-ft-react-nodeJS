API de Otras Funcionalidades

INICIO
Obtener coordenadas

Esta API permite realizar operaciones relacionadas con usuarios, como obtener usuarios, crear usuarios, actualizar usuarios y eliminar usuarios.

Ruta de la API: https://ncback-production.up.railway.app

Map Coordenadas

Envia una direccion y recibe las coordenadas

  • Método: POST
  • Ruta: /api/other/mapcord

Parámetros de consulta:

  • direccion (string, obligatorio): Direccion a buscar.

Ejemplo de petición: body (JSON)


  {
    "direccion": "Avenida 60 82, La Plata"
  }

Respuesta exitosa:


            {
              "lat": 4.59808,
              "lng": -74.076043,
              "direccion": "Avenida 60 82, La Plata, Buenos Aires Province, 1900, Argentina"
            }

Respuesta no exitosa:


            {
              "mensaje": "No se encontraron coordenadas"
            }


GET FAVORITOS

Visualiza el array de favoritos

  • Método: GET
  • Ruta: /api/favoritos

Parámetros de consulta:

  • id (param): id de usuario.

Ejemplo de petición:


  ruta/123456

Respuesta exitosa:


          {
            "fav": [
                "ponciano"
            ]
          }

Respuesta no exitosa:


            {
              "mensaje": 'Error al obtener favoritos'
            }

PUT FAVORITOS

Edita el array de favoritos

  • Método: PUT
  • Ruta: /api/favoritos

Parámetros de consulta: param

  • id (param): id de usuario.
  • data (array(string)): array de string de favoritos.

Ejemplo de petición:


      ruta/123456

Ejemplo de petición: (body)


          {
            "data":["ponciano"]
          }

Respuesta exitosa:


          {
            "msg": true
          }

Respuesta no exitosa:


    {
      "mensaje": 'Error al agregar favoritos'
    }
